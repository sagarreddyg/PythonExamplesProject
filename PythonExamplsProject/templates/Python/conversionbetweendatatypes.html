Conversion between data types

We can convert between different data types by using different type conversion functions like int(), float(), str() etc.

Conversion from int to other DataTypes.

    
        >> a = 4
        >> # converting from int to float
        >> b = float(a)
        >> print(a, b)
        >> 4 4.0
        >> # converting from int to string
        >> c = str(a)
        >> print(a, c)
        >> 4 4

We con't convert int to List,Tuple,Dictionaries and sets.because of integer object are not iterable

Conversion from Float to other DataTypes.

    
        >> f = 40.23
        >> # converting float to int
        >> x = int(f)
        >> print(x)
        >> 40
        >> # converting float to string
        >> y = str(f)
        >> print(y)
        >> 40.23

we can't convert float values to List,Tuple,Dictionaries and sets.because of float object are also not iterable
