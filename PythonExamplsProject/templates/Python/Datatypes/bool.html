Python Bool:

The Boolean data type can be either True or False. Booleans are used to represent the truth values that are associated with the logic branch of mathematics, which informs algorithms in computer science.

The values True and False will also always be with a capital T and F respectively, as they are special values in Python.

Lets see the below example:

We can perform many operations based on True or False on any Python DataType

    
        >> 1 > 2
        >> False
        >> 1 < 2
        >> True
        >> a = "abc"
        >> b = "def"
        >> print(a == b)
        >> False
        >> c = "abc"
        >> d = "abc"
        >> print(c == d)
        >> True
