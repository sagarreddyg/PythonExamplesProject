Python List:

List is an ordered sequence of items. It is one of the most used datatype in Python and is very flexible. All the items in a list do not need to be of the same type. Declaring a list is pretty straight forward. Items separated by commas are enclosed within brackets [ ].

    
        >> # List of integers looks like
        >> i = [1, 2, 3, 4]
        >> # list of flot values looks like
        >> f = [1.23, 3.4, 4.0, -23.2]
        >> # list of string values looks like
        >> s = ['abc', 'xyz', 'def']
        >> # list of list values looks like
        >> ll = [[1,2,3], ['abc', 'def'], [1.3, 4.6, -34.0]]

We can use the slicing operator [ ] to extract an item or a range of items from a list. Index starts form 0 in Python.

    
        >> a = [1, 2, 3, 4]
        >> # Printing list of values
        >> print(a)
        >> [1, 2, 3, 4]
        >> # getting list values by Positive index number
        >> print(a[0])
        >> 1
        >> print(a[1])
        >> 2
        >> print(a[:2])
        >> [1, 2]
        >> print(a[1:3])
        >> [2, 3]
        >> print(a[1:])
        >> [2, 3, 4]
        >>
        >>
        >> # getting list values by Negative index number
        >>
        >> print(a[-1])
        >> 4
        >> print(a[-4])
        >> 1
        >> print(a[:-3])
        >> 1
        >> print(a([-1:])
        >> 4
        >> print(a([-3:-1])
        >> [2, 3]
