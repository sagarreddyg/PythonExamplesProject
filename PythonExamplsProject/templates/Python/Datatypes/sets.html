Python Set:

Set is an unordered collection of unique, iterable and mutable objects. Set is defined by values separated by comma inside braces { }.

        >> my_set = {1,2,2,3,4,5}

We cannot access or change an element of set using indexing or slicing. Set does not support it. We can add single element using the add() method and multiple elements using the update() method.
