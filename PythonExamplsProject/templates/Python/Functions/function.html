Python Functions:

A function is a block of instructions that performs a specific task and,once defined, can be reused. Functions help break our program into small parts when our program grows larger and larger.it avoids repetition and makes code reusable.

Types of Functions:

In Python we have two types of functions

  • Built-in functions:Functions that are built into Python.
  • User-defined functions: Functions defined by the users themselves.

Python has a number of built-in functions:

  • print() which will print an object to the terminal.
  • int()which will convert a string or number data type to an integer data type.
  • len()which returns the length of an object.
