Getting started with Python:

Install Python on Windows Steps:

Installing and running Python on your personal computer is no difficult task. It involves just a few simple steps:

  • 1) First, download the latest version of Python 3 or Python 2 setup file from the official website python.org.
  • 2) Double-click on the file downloaded from official website.
  • 3) Add Python to system environment variables
  • 4) Install pip
  • 5) [Optional] Install virtualenv using pip
